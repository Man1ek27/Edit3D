# Edit3D
Do zbudowania aplikacji wystarczy sklonować repozytorium do folderu i otworzyć folder za pomocą Microsoft Visual Studio, wykorzystując opcję otwórz folder.
Program sam zauważy CMakeLists.txt i rozpocznie budowę. (Jeśli nie wykryje to można ręcznie otworzyć CMakeLists.txt i wtedy program zacznie budować.

## Testy

Testy manualne aplikacji zostały opisane w pliku `scene_full.txt`.
Zawierają one zestaw poleceń weryfikujących poprawność parsera, zarządzania obiektami, transformacji oraz zapisu i odczytu sceny.
